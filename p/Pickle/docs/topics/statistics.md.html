Statistics

This is a page to track some categories and log their size. This is used to create indicators for impact on the different projects.

To get numbers for base modules and the documentation and test cases, templates are added to the system messages scribunto-doc-page-show and scribunto-doc-page-does-not-exist. The templates will add the module to the actual categories, and then intersections can be made on those categories to get the actual numbers.

At nowiki those templates are w:no:grunnmoduler (base modules), w:no:grunnmoduler med dokumentasjon (base modules with documentation), and w:no:grunnmoduler med testtilfeller (base modules with test cases).

Use Category tree intersection to do further drill-down, for example «wmflabs:catscan3:grunnmoduler med testtilfeller and grunnmoduler med dokumentasjon».
