Info
 Domain: Domain of your wiki. Used for the URL bar. For example: wiki.me.org. Cannot be changed after wiki is created
 Name: Friendly name for the wiki. Shown in list of wikis. For example: My wiki
 Folder: Folder for the wiki. A file with the domain name will be created in the folder: For example: C:\personal\wiki.me.org. Note that this will create a file called C:\personal\wiki.me.org\wiki.me.org.xowa
 Main Page: The default start page for the wiki. For example, entering in wiki.me.org will navigate directly to Main_Page. Cannot be changed after wiki is created
For more info, see the new wiki help page
