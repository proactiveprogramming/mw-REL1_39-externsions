Notes
 Please make sure you are on Wi-Fi when downloading! Most wikis will be many MB or even GB.
 Download Central supports pause and resume. Pause the task (or close the app) and resume later.

Buttons
 Select a folder location for the wikis
 Download the latest list of wikis from archive.org
  Move a task to the queue
  (In the queue table) Remove a task from the queue. All work will be lost
  (In the imported table) Remove the task from imported and place it back in the available table
  Start the queue
  Pause the queue
  Restart a failed task from the last known step
