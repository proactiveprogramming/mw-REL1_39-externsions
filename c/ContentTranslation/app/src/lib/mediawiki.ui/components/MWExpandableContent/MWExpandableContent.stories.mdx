# ExpandableContent

MWExpandableContent provides a content panel that is by default
collapsed to a given size and can be optionally expanded to its maximum
content size by dragging a handler.

export const ExpandableContentWithScrollButtons = (args, { argTypes }) => ({
  components: { MwExpandableContent },
  props: Object.keys(argTypes),
  template: `
      <mw-expandable-content

        :scroll="scroll"

        <div>

});
